# CrashSense React E-Commerce Example

This example demonstrates how to integrate CrashSense into a real-world React e-commerce application.

## What This Example Shows

- Wrapping your app with `CrashSenseProvider`
- Using `useCrashSense` hook for manual error capture
- Adding breadcrumbs for user action tracking
- Integrating AI-powered crash analysis
- Sending crash reports to your backend
- Pre-crash warning system integration

## Setup

```bash
npm install @crashsense/core @crashsense/react @crashsense/ai
```

## Files

- `app.tsx` -- Main application with CrashSenseProvider setup
- `checkout.tsx` -- Checkout page with breadcrumbs, manual captures, and AI analysis
